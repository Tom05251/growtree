Flashcards PC Input

Flashcards PC Input is a PC input app for preparing flashcards and notes. It is provided as a Progressive Web App, so you can open it in Chrome or Edge and install it on your PC like an app.

This app does not use an EXE, MSI, or MSIX installer. Use Chrome or Edge to open the app and install it as a PWA.

Open App

Open the app from the button below. If possible, use Chrome or Edge on your PC.

Open Flashcards PC Input

Install on Chrome or Edge

  1. Open Flashcards PC Input in Chrome or Edge.
  2. Use the install icon near the address bar, or open the browser menu.
  3. Select "Install app" or "Install this site as an app".
  4. After installation, start it from the Start menu or desktop shortcut.

The exact menu text may differ depending on your browser and Windows version.

Updates

When a new version is available, the app shows an update notice after it opens. Press "Update now" to save your current input, check for the newest Service Worker, and reload the app.

Notes

  • The app checks for the latest version when an internet connection is available.
  • After the first load, the basic app screen can open offline.
  • If you use a shared PC, be careful with locally saved personal data.
